# Sample Data

Short inspection clips for testing the P1 agent. Video files are git-ignored — do not commit media; use the fetch script or drop your own files.

## How to populate
- Add licensed clip URLs to `../scripts/fetch_sample_data.sh` and run it, or
- Drop your own `.mp4` clips directly into this folder.

## Sources & licensing
Use only clips you have rights to: royalty-free stock (Pexels, Pixabay) under their licenses, NVIDIA VSS sample assets, or your own footage. Record each clip's source + license below.
